Ecocycle Discussion Tool — 共用儲存後端：Google 雲端硬碟。資料會直接寫入您指定的 Drive 資料夾，任何跨平台裝置只要登入有權限的 Google 帳號都能共編、下載與管理。

⚙️ 雲端連線設定（第一次使用請先完成，之後瀏覽器會記住）
尚未登入。請先貼上 Client ID 並儲存設定，再點擊登入。
  1. 前往 console.cloud.google.com 建立專案，於「API 和服務→已啟用的 API」啟用 Google Drive API。
  2. 「API 和服務→OAuth 同意畫面」：User type 選 External，填基本資料，新增範圍 drive（完整雲端硬碟存取，非 drive.file）、email、profile，並在「測試使用者」加入所有要共編的人的 Google 帳號 email。
  3. 「API 和服務→憑證→建立憑證→OAuth 用戶端 ID」，類型選「網頁應用程式」，「已授權的 JavaScript 來源」填入這個工具實際會被開啟的網址（例如 GitHub Pages 網址；本機測試可加 http://localhost:三位數埠號）。建立後複製 Client ID 貼到上方欄位。
  4. 把您的 Drive 資料夾分享權限改成「知道連結的人：編輯者」，或把協作者的帳號加入「編輯者」，否則登入後仍無法寫入檔案。
  5. 注意：Google 登入需要網頁有真實的 https 網址（不能是雙擊打開的本機檔案），建議把這個 HTML 上傳到 GitHub Pages 或 Netlify 後再使用。

🆕 建立新討論主題

輸入主題名稱，系統會在您的 Drive 資料夾產生一個對應檔案，並給您一組 6 位代碼分享給團隊。

🔑 加入既有討論主題

輸入對方提供的 6 位主題代碼（不分大小寫），系統會到同一個 Drive 資料夾找到對應檔案並載入。
